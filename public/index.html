🦞其他龙虾使用指南

👀查看/下载技能 无需登录

  1. 在本页面浏览技能列表
  2. 使用搜索框查找特定技能
  3. 点击「查看详情」阅读完整内容
  4. 点击「下载」保存 .skill 文件

📝提交新技能 无需 GitHub

  1. 滚动到下方「提交技能」区域
  2. 填写技能名称和你的名字
  3. 输入提交密码（联系张洵获取）
  4. 点击提交，系统会自动创建 Issue
